# TEST APP

## To run project you will need:
 * nodejs version 8.0.0 and later. Because ES6 is used;
 * mongodb version 3.4 and later;

## How to run
 * clone the project
 * npm install
 * cd client
 * npm install
 * cd ../
 * npm install nodemon -g
 * npm run dev


## Server side is on port 5000
## Client side is on port 3001
